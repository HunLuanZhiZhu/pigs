# 新建 pigs-api crate：server + phased_* 全部移走

## 目标

把 HTTP API 服务器（`server.rs`）和 6 个 phased 模块从当前位置移到新的 `pigs-api` crate。新 crate 所有代码逐行中英双语注释。

## 新 crate 结构

```
crates/pigs-api/
├── Cargo.toml
└── src/
    ├── lib.rs                  # 公开所有模块
    ├── server.rs               # 从 crates/pigs/src/server.rs 移入
    ├── phased_api_convert.rs   # 从 pigs-cli 移入
    ├── phased_runtime.rs       # 从 pigs-cli 移入
    ├── phased_phase.rs         # 从 pigs-cli 移入
    ├── phased_markers.rs       # 从 pigs-cli 移入
    ├── phased_tools.rs         # 从 pigs-cli 移入
    └── phased_prompts.rs       # 从 pigs-cli 移入（re-export pigs_prompts）
```

## 依赖关系（改后）

```
pigs-prompts  (无内部依赖，只依赖 pigs-config)
     ↑
pigs-api  (依赖 pigs-core, pigs-llm, pigs-config, pigs-tools, pigs-prompts, axum, tower-http)
     ↑
pigs-cli  (依赖 pigs-api + 其它；agent.rs 用 pigs_api::phased_runtime)
     ↑
pigs  (二进制，依赖 pigs-cli + pigs-api)
```

## 改动步骤

### 1. 创建 pigs-api crate
- `crates/pigs-api/Cargo.toml`：依赖 pigs-core, pigs-llm, pigs-config, pigs-tools, pigs-prompts + axum, tower-http, serde, serde_json, tokio, tracing, anyhow, thiserror, uuid, chrono, futures-util, async-trait, reqwest
- `crates/pigs-api/src/lib.rs`：`pub mod` 声明 7 个模块

### 2. 移动文件
- `crates/pigs/src/server.rs` → `crates/pigs-api/src/server.rs`
- `crates/pigs-cli/src/phased_*.rs`（6 个）→ `crates/pigs-api/src/phased_*.rs`
- 移入后 `use crate::phased_*` 不用改（crate 内部引用不变）
- `server.rs` 的 `use pigs_cli::phased_*` 改为 `use crate::phased_*`

### 3. 根 Cargo.toml
- members 加 `"crates/pigs-api"`
- workspace.dependencies 加 `pigs-api = { path = "crates/pigs-api" }`

### 4. pigs-cli 改动
- Cargo.toml 加 `pigs-api = { workspace = true }` 依赖
- `lib.rs`：删掉 6 个 `pub mod phased_*`，改为 `pub use pigs_api::phased_api_convert / phased_runtime / phased_tools / phased_prompts / phased_phase / phased_markers`（re-export，保持 `pigs_cli::phased_runtime` 等旧路径可用）
- `agent.rs`：`crate::phased_runtime` 不改（通过 re-export 仍可用）
- `phased_runtime.rs` 内的 `use crate::phased_markers` 等不用改（在 pigs-api 内部）

### 5. pigs crate 改动
- Cargo.toml 加 `pigs-api = { workspace = true }` 依赖
- `main.rs`：`pigs_cli::phased_runtime` 改为 `pigs_api::phased_runtime`（或通过 pigs-cli re-export 不改）
- `main.rs`：删掉 `mod server;`，改为 `use pigs_api::server`
- `server.rs` 不在 pigs crate 里了

### 6. 中英双语注释
- 所有移入 pigs-api 的文件，逐行加中英双语注释
- 包括文件级 `//!`、struct/fn 级 `///`、行内 `//`

## 验证
