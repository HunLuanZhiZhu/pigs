# 融合 mini-proxy 到 pigs：改名 + lib 化 + 前置路由 + 统一配置 + ×2 模型

## 架构（融合后）

```
客户端请求 (端口 3927)
    │
    ▼
pigs-proxy server (前置路由层，从 mini-proxy 改名+lib化)
    │
    ├── model 无 -pig 后缀 → 透传到上游 LLM（mini-proxy 原有逻辑）
    │
    └── model 有 -pig 后缀 → 调用 pigs-api 相位运行时 → 返回
```

## 步骤

### 1. mini-proxy → pigs-proxy：改名 + lib 化

- 将 `mini-proxy/` 源码移入 workspace 为 `crates/pigs-proxy`
- `Cargo.toml`：包名 `pigs-proxy`，加 `[lib] name = "pigs_proxy"`
- `src/main.rs` → `src/lib.rs`：`pub mod config/server/upstream/retry/protocol/log;`
- 暴露 `pub fn serve(config, port)` 供 pigs 二进制调用
- 删掉 `main.rs` 里的 main 函数（二进制入口移到 pigs crate）

### 2. 统一配置：mini-proxy config.toml + 顶层 pigs 字段

在 mini-proxy 的 `Config` 顶层加 pigs 需要的字段：

```toml
# mini-proxy 原有
[server]
listen = "127.0.0.1:3927"   # 改默认端口
clean_empty_content = true

[log]
level = "info"
...

[[provider]]
name = "..."
models = ["xopglm52", "auto", ...]
...

# pigs 新增顶层字段
language = "zh"              # 语言（zh/en）
permission_mode = "workspace_write"
max_turns = 50
max_tokens = 4096
temperature = 0.2
system_prompt = ""           # 可选自定义系统提示
compact_token_threshold = 100000
compact_keep_recent = 10
```

`Config` struct 加这些字段（带 `#[serde(default)]`）。

### 3. pigs-proxy server 增加 -pig 路由

`server.rs` 的 `handle` 函数里，在 `pick_endpoint` 之前加判断：

```rust
if client_model.ends_with("-pig") {
    // 剥离 -pig 后缀，走 pigs-api 相位运行时
    let real_model = client_model.trim_end_matches("-pig");
    // 调用 pigs_api::run_converted_turn(...)
    // 返回相位化结果（按请求格式构造响应）
} else {
    // 原有透传逻辑
}
```

pigs-proxy 依赖 `pigs-api`。

### 4. ×2 模型 ID

`/v1/models` 端点：遍历所有 provider 的 `models` 列表，每个模型输出两条：
- `{model}` — 透传
- `{model}-pig` — 相位化

### 5. pigs crate 适配

- `main.rs`：不再直接用 `pigs_api::server::serve`
- 改为加载 `pigs_proxy::Config`，调用 `pigs_proxy::serve(config, runtime)`
- `runtime`（PhasedRuntime）传给 pigs-proxy 供 -pig 路径使用

### 6. pigs-config 适配

- `AppConfig` 改为从 mini-proxy 格式 `Config` 读取
- 或：pigs-proxy 的 `Config` 加便捷方法 `to_app_config()` 转换
- 保留 `resolve_model` / `ResolvedModel` 供 CLI agent 用

## 文件改动清单

| 文件 | 改动 |
|---|---|
| `crates/pigs-proxy/` | 新 crate，从 mini-proxy 移入 + lib 化 + 改名 |
| `crates/pigs-proxy/Cargo.toml` | 包名 pigs-proxy，加 [lib]，依赖 pigs-api |
| `crates/pigs-proxy/src/lib.rs` | 从 main.rs 改写，pub mod + serve 函数 |
| `crates/pigs-proxy/src/config.rs` | Config 加顶层 pigs 字段 |
| `crates/pigs-proxy/src/server.rs` | handle 加 -pig 路由分支 + /v1/models ×2 |

| `crates/pigs/config.toml` | 统一配置文件（mini-proxy 格式 + pigs 字段）|
